Using AI To Raise Money For Charity

Being one of my more straightforward projects, this one is something that I am very proud of. Using ComputerVision, I was able to create a program that can read a question from a charity website and answer the questions accordingly. On this particular website, every question you get correct acts as a donation to the UN World Food Programme. See, AI can be used for good!
